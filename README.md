# REST_API

Simple Express + EJS example. Start the app locally:

```bash
npm install
node index.js
# then open http://localhost:8080/post
```

Deploy notes:
- To push to GitHub: create a repo, then `git remote add origin <url>` and `git push -u origin main`.
- To deploy on Vercel: connect the GitHub repo in the Vercel dashboard or use the `vercel` CLI.
